# 🚀 Local Setup

Clone the repository

git clone https://github.com/AnuradhaKashaudhan/Portfolio.git


Navigate into the project


cd Portfolio


Install dependencies


npm install


Run the development server


npm run dev


Application will run at:


http://localhost:5173
